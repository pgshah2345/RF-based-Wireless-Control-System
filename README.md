# RF-Based Wireless Control System

## Overview

This project demonstrates a long-range wireless control system using a 433 MHz RF Transmitter and Receiver module. The system enables wireless switching of both AC and DC loads through a relay driver circuit without using any microcontroller.

The transmitter sends encoded RF commands which are received and decoded by the receiver. The receiver then activates the relay module to control connected loads.

---

## Features

- Wireless control of AC and DC appliances
- Operating range up to 50 meters
- No microcontroller required
- Low-cost implementation
- Simple transmitter-receiver architecture

---

## Components Used

### Transmitter Side
- 433 MHz RF Transmitter
- Encoder IC (HT12E)
- Push Buttons
- Battery Supply

### Receiver Side
- 433 MHz RF Receiver
- Decoder IC (HT12D)
- Relay Driver Circuit
- Relay Module
- AC Bulb
- DC LED Load

---

## Working Principle

1. User presses a button on the transmitter.
2. Encoder IC converts the command into encoded data.
3. RF transmitter sends the data wirelessly.
4. RF receiver captures the signal.
5. Decoder IC decodes the received command.
6. Relay driver activates the relay.
7. Connected AC/DC load turns ON or OFF.
